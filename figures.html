1 Figures and tables

1.1 Layout

Review comments received from PNAS indicated that the environmental data and results from elemental analysis (Fig. S6) were the most interesting contributions of the manuscript. Let’s consider restructuring the text to emphasize these results.

Introduction needs to describe lack of in situ data on environmental conditions that are relevant for evaluating larval response to low pH conditions - it hints at this but should be elaborated. Also emphasize that we do not have a consistent methodology for measuring larval response to low pH, i.e., we need a consistent methodology for measure dissolution. This will setup the rest of the text to highlight our environmental results and how the method for measuring dissolution is novel and leads to more insight (e.g., elemental results).

  1. Figure: study sites map
  2. Dissolution results: need to combine old figures 2, 4, S1, S2 in a way to show what information is obtained with the new method, reviewers commented that the info provided by each figure was unclear
  3. Elemental mapping: old Figure S6 but need to present more clearly
  4. Exploratory RDA plot for bio response and environmental variables
  5. Environmental data: maps, PCA, gradients, by depth (old S3 but with clarity)
  6. Correlation matrices: environmental v biological data
  7. Regression plots: dissolution measures vs each other (e.g., how is external related to internal), dissolution measures vs environmental gradients
  8. Regression plots: dissolution vs carapace width

Tables can also be provided to support regression results in the figures.

totab <- envdatdelt %>% 
  mutate(
    val = ifelse(var %in% 'Chla', 10 ^ val, val) # back-transform chla for table
  ) %>% 
  filter(CTD %in% dissdat$CTD) %>% 
  filter(depth %in% 60) %>% 
  left_join(shoreloc, by = 'CTD') %>% 
  select(-depth) %>% 
  group_by(var, shoreloc) %>% 
  summarise(
    avev = mean(val, na.rm = T), 
    medv = median(val, na.rm = T), 
    minv = min(val, na.rm = T), 
    maxv = max(val, na.rm = T)
  ) %>% 
  ungroup %>% 
  gather('varv', 'valv', -var, -shoreloc) %>% 
  mutate(
    valv = ifelse(var %in% c('Oxygen', 'Temperature', 'pCO2'), round(valv, 1), round(valv, 2)),
    valv = as.character(valv)
  ) %>% 
  spread(varv, valv) %>% 
  arrange(desc(shoreloc), var) %>% 
  mutate(
    shoreloc = factor(shoreloc, levels = c('onshore', 'offshore'), labels = c('Onshore', 'Offshore')),
    shoreloc = as.character(shoreloc),
    shoreloc = ifelse(duplicated(shoreloc), '', shoreloc)
  ) %>% 
  dplyr::select(shoreloc, var, avev, medv, minv, maxv) %>% 
  rename(
    Location = shoreloc,
    Parameter = var,
    Average = avev,
    Median = medv,
    Minimum = minv,
    Maximum = maxv
  )

crs <- crossing(var1 = names(dat_cor), var2 = names(dat_cor)) %>% 
  filter(var1 != var2) %>% 
  rownames_to_column() %>% 
  group_by(rowname) %>% 
  nest %>% 
  mutate(
    crs = map(data, function(x){
      
      # variables
      vr1 <- dat_cor[[x$var1]]
      vr2 <- dat_cor[[x$var2]]
      
      # pearson
      pr_ts <- cor.test(vr1, vr2, method = 'pearson')
      pr_cr <- round(pr_ts$estimate, 2)
      pr_pv <- p_ast(pr_ts$p.value)
      pr <- paste(pr_cr, pr_pv)
    
      out <- data.frame(pr = pr, stringsAsFactors = F)
      return(out)
      
    })
  ) %>% 
  unnest %>% 
  select(-rowname)

levs <- names(dat_cor)
